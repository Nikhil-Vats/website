      Find Datasets

      Find the data you need quickly (preferably in a single place).

      Before getting to insights, you first need to find good datasets.

      Context

      Often, it's one of the first requirements of data projects.

      Initially, you may have some datasets at hand. Over time, you will want to have good references already saved as favorites.

      # Examples

      Google: “Rainfall in the Amazon CSV”

      # Solutions

      DataHub
      Created by the original authors of Frictionless Data, it’s a place for finding datasets of public interest. Everything is containerized using Data Packages.
